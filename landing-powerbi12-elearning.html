Temario del curso

Revisarás conexión, transformación, modelamiento, DAX y dashboards en una ruta que combina fundamentos técnicos con aplicación práctica a reportes y análisis.

  • Introducción a Power BI Desktop y Power BI Web
  • Diferencias entre Excel, Web y Desktop
  • Fundamentos de Business Intelligence y reporting
  • Exploración general de la interfaz y flujo de trabajo
  • Importación de datos desde TXT, CSV, Excel, Access y otras fuentes
  • Limpieza y transformación con el editor de consultas
  • Introducción a Power Query y preparación del modelo
  • Enriquecimiento de datos para análisis posteriores
  • Introducción al modelo entidad-relación
  • Tipos de relaciones y lógica entre tablas
  • Tablas maestras, transacciones y estructura del modelo
  • Buenas prácticas para construir modelos consistentes
  • Creación de medidas y columnas calculadas
  • Estructura general de funciones y lógica de cálculo
  • Uso de SUM, SUMX, CALCULATE, FILTER y VALUES
  • Funciones de fecha, condición y relación como DATEADD, IF y RELATED
  • Profundización en medidas avanzadas y contexto de cálculo
  • Distinct Count, filtros avanzados y lógica de segmentación
  • Optimización de fórmulas y lectura de resultados
  • Aplicación de DAX a indicadores y reportes analíticos
  • Tablas, matrices, segmentadores, filtros y gráficos
  • Visualizaciones como mapas, gráficos circulares, treemaps y dashboards temáticos
  • Formato visual: títulos, textos, fondos, colores y diseño
  • Construcción y presentación de un dashboard de mayor complejidad
