# HTML_ESTATICO Design Tokens

## Tipografía
- Fuente principal: `Inter`, `Segoe UI`, `Roboto`, `Arial`, `sans-serif`
- Tamaños principales detectados:
  - Titulares hero: `clamp(32px, 5vw, 52px)`
  - Subtítulos: `20px`
  - Texto normal: `14px` - `17px`

## Colores principales
- Azul profundo: `#004A99`, `#005596`, `#0078D4`, `#0f254a`
- Amarillo / dorado: `#ffca28`, `#ff9800` y variantes
- Blanco: `#ffffff`
- Grises: `#64748b`, `#cbd5e1`, `#e2e8f0`
- Fondo oscuro: `#08111c`, `#0d1727`
- Sombras suaves y degradados de referencia tech

## Espaciado y layout
- Padding hero: `60px 10%`
- Card padding: `22px`, `45px`, `35px`
- Gap entre secciones: `30px` - `50px`
- Ancho máximo del contenido: `max-width: 1200px`, `max-width: 1000px`

## Botones y UI
- Botón primario: `border-radius: 50px`, `padding: 14px`, `font-weight: 900`
- Botones de propuesta: colores `#ff9800` / `#0288d1`
- Tarjetas con fondo blur y bordes redondeados: `border-radius: 24px` / `25px`

## Tokens funcionales
- `primary`: `#1c4587` / `#004A99`
- `secondary`: `#eef4fb`
- `accent`: `#ff9800`
- `dark`: `#0f254a`
- `darkCard`: `#16325c`

## Diseño de iconografía y badges
- Badges redondas con fondo semitransparente
- Íconos embebidos como SVG en CSS para estabilidad visual
- Uso de gradientes para hero y tarjetas

## Nota
